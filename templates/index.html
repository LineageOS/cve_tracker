Welcome

What is this?

The CVE tracker is a manual way to keep track of the patch status of CVEs for the kernels used in official devices

What's a CVE?

Common Vulnerabilities and Exposure. In a few words: These are basically security holes which can (mostly not even that) be used to get access to your device's data

The kernel for my device is only at x%, why is it so insecure?

First, please note that the tracker has to be manually maintained by each device maintainer. It does not necessarily represent the actual patch status of the kernel. Additionally, only very few of the CVEs can actually be used to exploit your data, others are just theoretical security holes and can't be used due to other security measures such as SELinux
